######################################################################################################## --- Bienvenue sur notre documentation ! --- ######################################################################################################## Merci d'utiliser notre logiciel. Voici la documentation officielle dans laquelle vous pouvez trouver le détails des fonctionnalitées du logiciel. En espérant que cela puisse vous aider :) * Fonctionnalité numéro 1: Créer une base de données * Fonctionnalité numero 2: Créer une table dans une base de données. Vous pouvez effectuer cette actions après avoir creér votre base de données. * Fonctionnalité numéro 3: Renommer une table. En cas d'erreur lors de la création de votre table, vous avez la possibilité de la renommer. * Fonctionnalité numéro 4: Ajouter une colonne a une table. Si vous le souhaitez, ou en cas d'oubli, vous avez la possibilité d'ajouter une colonne a une table de votre base de données. * Fonctionnalité numéro 5: Renommer une colonne d'une table. Si vous le souhaitez, ou en cas d'erreur, vous avez la possibilité de renommer une colonne de votre base de données. * Fonctionnalité numéro 6: Remplir une table. Ajouter les données de votre choix dans une table. * Fonctionnalité numéro 7: Supprimer une base de données. Si vous le souhaitez, et pour n'importe quel raison vous avez la possibilités de supprimer une base de données. * Fonctionnalité numéro 8: Supprimer une ligne d'une table. Si vous le souhaitez, ou en cas d'erreur, vous avez la possibilité de surpprimer une ligne d'une table. * Fonctionnalité numéro 9: Supprimer une colonne d'une table. Si vous le souhaitez, vous avez la possibilité de supprimer une colonne d'une table, bien sur a condition que ce ne soit pas la colonne contenant la clé primaire. * Fonctionnalité numéro 10: Supprimer une table. Si vous le souhaitez, ou en cas d'erreur, vous avez la possibilité de supprimer une table d'une base de donnée. * Fonctionnalité numéro 11: Affichage des résultats. * Fonctionnalité numéro 12: Afficher les bases de données contenues dans le serveur mysql de notre logiciel. * Fonctionnalité numéro 13: Afficher les tables d'une base de données. * Fonctionnalité numéro 14: Exporter des données d'une table. Il est possible pour vous de faire un export sous forme de tableau organisé des données contenues dans la table de votre choix. * Fonctionnalité numéro 15: Consulter la documentation au préalable récupérée avec CURL. ######################################################################################################## Une question ?
